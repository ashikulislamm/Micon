# Service Images Guide

Add the following images to your `/public/services/` directory:

## Required Images

1. **peb-steel.jpg** - PEB Steel Buildings
   - Industrial steel building construction
   - Factory or warehouse with steel framework
   - Suggested size: 1200x800px

2. **rcc-structures.jpg** - Turnkey RCC Structures
   - RCC building under construction
   - Concrete structure or high-rise building
   - Suggested size: 1200x800px

3. **etp-wtp.jpg** - ETP/WTP Facilities
   - Industrial water treatment facility
   - Treatment plant equipment or tanks
   - Suggested size: 1200x800px

4. **green-construction.jpg** - Sustainable Green Construction
   - Modern eco-friendly industrial building
   - Green construction site with environmental features
   - Suggested size: 1200x800px

## Directory Structure
```
public/
  services/
    peb-steel.jpg
    rcc-structures.jpg
    etp-wtp.jpg
    green-construction.jpg
```

These images will be automatically displayed in the Expertise section cards with hover effects.
